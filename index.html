Magic Crystals

Use the crystals to power up the arcane core. Be careful though, overpowering it could be dangerous!

Use the four crystals to power up arcane cores. Each core has a set amount of power it can handle and you must be careful not to overpower it.

Each of the crystals has a set amount of power it can distribute. Clicking on the crystal charges the core by that amount each time.

To win the game, charge the core up to the exact number, over-charging the core means you lose. Either way, the game resets and the values of the core and crystals change and you can try again.
